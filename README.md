# Green Wheels - Core

Common logic for Green Wheels Project.

## Usage
For using with the monorepo refer to the root project [**Green Wheels**](https://github.com/lysandre995/green-wheels)


Standalone usage:

* Install dependency:
```bash
    npm i
```

* Compile the project:

```bash
    npm run build
```

* Compile the project and watch for changes:

```bash
    npm run watch
```

* Run the project:

```bash
    npm run start
```
