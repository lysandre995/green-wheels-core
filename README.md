# Green Wheels - Core

Common logic for theGreen Wheels Project

## Usage

Compile the project:
```bash
    npm run build
```
Compile the project and watch for changes:
```bash
    npm run watch
```
Run the project:
```bash
    npm run start
```
Run the tests:
```bash
    npm run test
```
Run the linter:
```bash
    npm run lint
```
Fix the linter errors:
```bash
    npm run lint-fix
```
